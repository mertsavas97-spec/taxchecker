# TaxChecker

TaxChecker is an SEO-first web platform that helps users estimate and understand U.S. federal taxes through fast, easy-to-use calculators and educational content.

Built with **Next.js**, **TypeScript**, and **Vercel**, the project focuses on performance, accessibility, responsive design, and search engine optimization.

## Live Demo

🌐 https://taxchecker.app

## Features

* Federal tax calculators
* SEO-optimized content structure
* Built-in blog and CMS
* Responsive design
* Dynamic metadata and Open Graph support
* Structured data (Schema.org)
* Google Analytics 4 integration
* Fast static generation and optimized performance

## Tech Stack

* Next.js (App Router)
* React
* TypeScript
* Tailwind CSS
* Vercel
* Google Analytics 4

## Project Goals

* Provide simple and accurate tax estimation tools
* Deliver fast page loads and excellent Core Web Vitals
* Build scalable calculator infrastructure
* Support long-term SEO growth through structured content

## AI-Assisted Development

This project was developed using an AI-assisted workflow with Cursor.
