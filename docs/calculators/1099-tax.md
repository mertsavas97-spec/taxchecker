# 1099 Tax Calculator — Specification

**Calculator ID:** `tax-1099`  
**Route (future):** `/calculators/1099-tax`  
**Engine module:** `src/lib/tax-engine/calculators/tax-1099.ts`  
**TaxChecker version:** 1.0

---

## Purpose

Estimate federal taxes owed on 1099-NEC / 1099-MISC independent contractor income. Frames the calculation around gross 1099 income and business expenses (or net income), producing SE tax and income tax estimates familiar to gig workers and freelancers.

**Engine behavior:** Delegates to the same core pipeline as Self Employed Tax Calculator. This calculator differs in UX framing, default copy, and optional gross-to-net expense path.

---

## Target User

- Independent contractors who received Form 1099-NEC
- Gig economy workers (rideshare, delivery, freelance platforms)
- Users searching immediately after receiving a 1099 in January
- Side-income earners combining W-2 and 1099 income

## Required Inputs

| Field | Type | Description |
|---|---|---|
| `taxYear` | `number` | Tax year |
| `filingStatus` | `FilingStatus` | Filing status |
| `incomeInputMode` | `'net' \| 'gross'` | Whether user enters net profit or gross minus expenses |
| `netBusinessProfit` | `Cents` | Required if mode = `net` |
| `gross1099Income` | `Cents` | Required if mode = `gross` |
| `businessExpenses` | `Cents` | Required if mode = `gross`; deducted from gross |

## Optional Inputs

| Field | Type | Default | Description |
|---|---|---|---|

## Formula Dependencies

```
if incomeInputMode === 'gross':
  netBusinessProfit = max(0, gross1099Income - businessExpenses)
// Then identical to self-employed-tax pipeline
remainingTaxOwed = max(0, totalFederalTax - federalWithholding)
```

**Engine modules:** Same as `self-employed-tax` (shared pipeline)

## Assumptions

1. 1099 income is self-employment income subject to SE tax (not statutory employee misclassification analysis).
2. Business expenses are ordinary and necessary — user-supplied; not validated.
3. No backup withholding tracked separately.
4. Multiple 1099s are summed by user into single gross/net figure.
5. 1099-K income treated same as other 1099 self-employment income in 1.0 (no separate threshold logic).

---

## Exclusions

- Employee vs contractor classification determination
- State income tax
- QBI deduction, AMT, credits (same as self-employed spec)
- Quarterly due dates (link to Quarterly calculator)
- Expense categorization or audit risk

---

## Edge Cases

| Case | Handling |
|---|---|
| Gross mode with expenses > gross | netBusinessProfit = 0; warning "expenses exceed income" |
| User has both W-2 and 1099 | w2Wages field coordinates SS wage base and adds to income tax |
| federalWithholding > totalFederalTax | remainingTaxOwed = 0; informational note about refund possibility |
| Zero 1099 income | Valid; all taxes from otherIncome if any |

---

## Validation Rules

| Field | Rule | Error code |
|---|---|---|
| `gross1099Income` | >= 0 when mode = gross | `OUT_OF_RANGE` |
| `businessExpenses` | >= 0 when mode = gross | `OUT_OF_RANGE` |
| `netBusinessProfit` | >= 0 when mode = net | `OUT_OF_RANGE` |
| Cross-field | At least one income path complete | `REQUIRED` |

All rules from `self-employed-tax` also apply.

## Disclaimer Copy

> **Estimate only — not tax advice.** This calculator estimates federal taxes on independent contractor income using IRS self-employment tax rules. Receiving a 1099-NEC generally means income is reported to the IRS; your actual tax depends on your full return, deductions, and credits. TaxChecker does not determine whether you are correctly classified as a contractor or employee. Consult a tax professional for advice.
