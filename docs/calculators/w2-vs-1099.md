## Purpose

Compare estimated **net take-home** and **total tax burden** between two scenarios at the same gross compensation level: (A) working as a W-2 employee and (B) working as a 1099 independent contractor / sole proprietor. Highlights the self-employment tax difference and employer payroll tax cost.

---

## Target User

- Workers evaluating job offers (employee vs contractor)
- Freelancers considering W-2 conversion
- Employers modeling contractor vs employee cost (optional employer view)
- Users searching "is 1099 better than W2"

## Formula Dependencies

```
// 1099 path (self-employed pipeline)
netBusinessProfit = grossCompensation - businessExpenses
// → SE tax + income tax (full pipeline)

// W-2 path
w2EmployeeFICA = computeEmployeeFICA(grossCompensation, config.payrollTax)
taxableIncomeW2 = max(0, grossCompensation - standardDeduction)
w2FederalIncomeTax = computeFederalIncomeTax(taxableIncomeW2, filingStatus)
w2TotalEmployeeTax = w2EmployeeFICA + w2FederalIncomeTax
w2NetTakeHome = grossCompensation - w2TotalEmployeeTax  // simplified; no state

// Employer cost (optional)
w2EmployerFICA = computeEmployerFICA(grossCompensation, config.payrollTax)
```

**Engine modules:** `payroll-tax`, `self-employment-tax`, `deductions`, `federal-income-tax`

## Assumptions

1. **Same gross number** for both scenarios — no rate normalization for contractor overhead.
2. W-2 scenario: employee pays employee FICA only; employer FICA shown only if `showEmployerCost`.
3. No benefits valuation (health insurance, 401k match, PTO) — excluded in 1.0.
4. 1099 scenario: user responsible for full SE tax (both halves).
5. Federal income tax only; standard deduction for both.
6. W-2 withholding estimate equals tax liability (no refund/owe delta) unless rate override.
7. Does **not** determine legal worker classification.

---

## Exclusions

- Benefits and perks monetary value
- State income tax
- Unemployment insurance, workers comp (except future employer cost)
- QBI deduction
- S Corp election path (see LLC vs S Corp)
- Misclassification legal analysis

## Edge Cases

| Case | Handling |
|---|---|
| businessExpenses = gross | 1099 net profit = 0; only W-2 comparison meaningful |
| Low income below standard deduction | Income tax = 0; FICA/SE still applies |
| gross above SS wage base | FICA and SE SS portions capped appropriately |
| showEmployerCost = true | Show separate employer cost card |

## Disclaimer Copy

> **Estimate only — not tax advice.** This comparison uses simplified federal tax assumptions and equal gross compensation. It does not evaluate employee benefits, legal worker classification, state taxes, or business structure options like S corporations. Being classified as a 1099 contractor when you are legally an employee has serious tax and legal implications. Consult an employment attorney or tax professional before making decisions.
